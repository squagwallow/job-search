# Writing Samples — Bank

Approved cover letters, Upwork proposals, and application-question responses. One sample per markdown file, following `formats/writing-sample-format.md`.

## What to add
Any cover letter, proposal, or application response you've submitted and would be comfortable using as reference or starting point for a similar future application.

## Filename convention
`[YYYY-MM-DD]-[channel]-[short-slug].md`

- `channel` is `upwork` or `general`.
- Example: `2026-03-12-upwork-notion-rework.md`

## File contents
See `formats/writing-sample-format.md` for required frontmatter and body structure.

## Why tags matter
The prepare-application prompt filters this bank by `channel` and `job_type` tags to find the most relevant samples for a listing. Consistent tagging is what makes the bank useful — not volume.

## POPULATED BY
You, by adding files here. Start with 3–5 of your best approved samples across the job types you commonly pursue. Add over time.
