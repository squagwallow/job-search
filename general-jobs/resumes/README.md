# Resume Variants

Your different resume versions live here. No rigid documentation process — swap freely per job.

## Suggested filename convention
`resume-[variant-slug].md` or `resume-[variant-slug].pdf`

Variant slug should describe the angle, not the job type alone. Examples:
- `resume-ai-prompt-engineer.md`
- `resume-clinical-care-coord.md`
- `resume-notion-consultant.md`
- `resume-generalist-tech.md`

## How the prepare-application prompt uses these
When Step 2 (profile comparison) runs, it may identify a best-fit resume variant and recommend it. Action items in Step 5 may include "tweak variant X" rather than drafting a new one.

## POPULATED BY
You, by dropping files here. No required frontmatter — the variant filename is the metadata.
