# Upwork Portfolio

**[STUB — populate with your portfolio items. See POPULATED BY at the end.]**

## Format — one entry per item

```
### [Item title]
- Link: [URL to item on Upwork or elsewhere]
- Short description: [1–2 sentences — what the project was]
- Role: [what you did]
- Tools / stack: [relevant tools]
- Outcome: [result, metric, or reference to a testimonial]
- Tags: [ai-prompt-engineering, notion, clinical, etc.]
- Strongest for: [1–2 sentences — what kind of listing this is the best response to]
```

## Why "strongest for" matters
The prepare-application prompt uses this field to pick which portfolio items to lead with when evaluating a listing. Be specific — not "good for tech roles" but "good for clients rebuilding complex Notion workspaces they've outgrown."
